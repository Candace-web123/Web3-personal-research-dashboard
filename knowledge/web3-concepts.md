## 一、BTC 周期六阶段

| 阶段 | 英文常量 | 特征 | 操作倾向 |
|---|---|---|---|
| 底部积累 | `BottomAccumulation` | 价格长期横盘，MVRV < 1，NUPL 处于投降区，恐惧指数极度恐惧 | 以观察为主，适合定投 |
| 上涨早期 | `EarlyUptrend` | 价格突破长期均线，MVRV 回升至 1-2，NUPL 进入乐观区 | 可小幅跟踪机会 |
| 上涨中期 | `MidUptrend` | 趋势确认，MVRV 2-3，NUPL 信念区，市场情绪转为贪婪 | 持有，关注过热信号 |
| 过热 | `Overheated` | MVRV > 3，NUPL 进入狂热区，恐惧贪婪 > 80，山寨币普涨 | 不建议新增仓位 |
| 顶部风险 | `TopRisk` | 价格加速冲顶后回落，多项指标背离，ETF 资金流入减速或转流出 | 建议降低风险暴露 |
| 下跌/防守 | `DowntrendDefensive` | 价格跌破关键均线，MVRV 回落，市场情绪转为恐惧 | 防守为主，观察底部信号 |

### 关键链上指标解释

| 指标 | 全称 | 含义 | 数据源 |
|---|---|---|---|
| MVRV | Market Value to Realized Value | 市值/已实现市值比值。>3 偏高，<1 偏低 | Glassnode / LookIntoBitcoin |
| NUPL | Net Unrealized Profit/Loss | 未实现盈亏净值。范围 −1 到 +1，>0.75 狂热，<0 投降 | Glassnode |
| Puell Multiple | Puell Multiple | 矿工收入与年均值比值。>4 过热，<0.5 底部 | LookIntoBitcoin |
| Pi Cycle | Pi Cycle Top Indicator | 111 日均线与 350 日均线的 2 倍值交叉，用于判断顶部 | LookIntoBitcoin |
| 200W MA | 200-Week Moving Average | 200 周均线，历史底部参考线 | Glassnode |
| 恐惧贪婪指数 | Fear & Greed Index | 0-100 综合情绪指标。0-25 极度恐惧，75-100 极度贪婪 | Alternative.me |

## 二、市场环境四分类

| 分类 | 英文常量 | 特征 | 五维评分倾向 |
|---|---|---|---|
| 强进攻 | `StrongRiskOn` | 流动性充裕，资金积极进入风险资产，TVL 上升 | 总分偏高（+3～+5） |
| 中性轮动 | `NeutralRotation` | 资金在板块间轮动，无明确方向性信号 | 总分中性（−1～+2） |
| 谨慎 | `Cautious` | 部分指标走弱，流动性边际收紧，个别板块出现风险 | 总分偏负（−2～0） |
| 防守 | `Defensive` | 流动性紧张，资金撤出风险资产，宏观环境恶化 | 总分偏低（−5～−2） |

### 五维评分维度

| 维度 | 含义 | 正分（+1） | 零分（0） | 负分（−1） |
|---|---|---|---|---|
| 稳定币流动性 | 稳定币总市值变化趋势 | 市值上升 | 持平 | 市值下降 |
| DeFi TVL | 全链 TVL 变化 | 上升 | 持平 | 下降 |
| DEX 成交量 | 去中心化交易所交易量 | 放量 | 正常 | 缩量 |
| ETF/机构资金 | BTC/ETH ETF 资金流入 | 净流入 | 持平 | 净流出 |
| 宏观政策 | 美联储政策/宏观环境 | 偏宽松 | 中性 | 偏紧缩 |

## 三、资产分层（S/A/B/C/D）

| 层级 | 含义 | 典型资产 | 是否参与 Alpha 评分 |
|---|---|---|---|
| S | BTC 周期锚点 | BTC | 否（独立评估） |
| A | ETH / 核心主流 | ETH | 部分参与 |
| B | 稳定币 / 流动性资产 | USDC、USDT、DAI | 否（`excludeFromAlphaScoring: true`） |
| C | 主流板块资产 | SOL、LINK、ARB 等 | 是 |
| D | 热点 / Alpha 候选 | 新项目、高增长协议 | 是 |

## 四、Alpha 评级与生命周期

### 评级（A/B/C/D）

| 评级 | 标准 |
|---|---|
| A | 赛道上升期 + 产品数据强 + 代币有真实价值传导 + 风险可控 |
| B | 多项正面但有一项待验证（如代币传导不确定） |
| C | 叙事强但数据弱，或数据强但代币价值捕获不清晰 |
| D | 高风险投机。仅跟踪，不建议配置 |

> 约束：A 级 Alpha 的代币传导强度不得为 `Weak` 或 `None`（由 `lib/data-guards.ts` 校验）

### 生命周期

| 状态 | 含义 | 下一步 |
|---|---|---|
| `NewlyFound` | 新发现，初步了解中 | 建立基础信息，观察 1-2 周 |
| `Watching` | 观察中，跟踪关键数据 | 等待数据变化或催化事件 |
| `FocusTracking` | 重点跟踪，数据走强 | 深挖团队/代币/竞品 |
| `RiskElevated` | 风险升高，需重新评估 | 确认风险是否可持续，考虑降级 |
| `Removed` | 移除出池 | 记录移除原因，存档备查 |

## 五、资产池内状态

| 状态 | 含义 |
|---|---|
| `Normal` | 普通观察，无异常 |
| `Moving` | 出现异动（价格/成交量/TVL 异常变化） |
| `AlphaCandidate` | 符合 Alpha 候选标准 |
| `DeepDive` | 重点深挖中 |
| `RiskElevated` | 风险升高，需关注 |
| `Removed` | 已移除出观察宇宙 |

## 六、强链分类

| 类型 | 英文常量 | 判定标准 |
|---|---|---|
| 资金沉淀型 | `CapitalDeposit` | TVL 高且增长，稳定币净流入 |
| 交易活跃型 | `TradingActive` | DEX 成交量/TX 数持续增长 |
| 应用收入型 | `AppRevenue` | 协议收入/Fees 排名靠前且增长 |
| 激励驱动型 | `IncentiveDriven` | 增长由激励计划驱动，需验证可持续性 |

## 七、代币价值传导

| 传导类型 | 含义 | 示例 |
|---|---|---|
| `CashFlow` | 协议收入分配给持币者（分红/回购） | GMX |
| `Buyback` | 协议用收入回购代币 | HYPE |
| `Usage` | 代币作为网络使用费（gas/质押） | ETH |
| `GovernanceExpectation` | 治理权预期价值（不一定有现金流） | UNI |
| `None` | 无明显价值传导 | WLD（治理+身份，无直接现金流） |

| 传导强度 | 含义 |
|---|---|
| `Strong` | 传导路径清晰，数据可验证 |
| `Medium` | 有传导路径但规模有限 |
| `Weak` | 传导路径不清晰或极少量 |
| `None` | 无可识别的价值传导 |
| `Uncertain` | 数据缺失，无法判断 |

## 八、风险标签体系

| 优先级 | 含义 | 响应要求 |
|---|---|---|
| P0 | 立即关注：安全漏洞、协议被攻击、团队跑路 | 当日评估是否需要减仓/退出 |
| P1 | 高优先级：监管风险、代币解锁高峰、数据异常 | 24h 内关注 |
| P2 | 中优先级：赛道叙事降温、竞品崛起 | 每日跟踪 |
| P3 | 低优先级：长期跟踪事项 | 定期复盘 |
